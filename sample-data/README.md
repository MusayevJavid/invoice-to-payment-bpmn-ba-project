# Sample Data

This folder will contain the synthetic dataset for the Invoice-to-Payment Analytics Extension.

The data will be created only for portfolio demonstration and analysis purposes. It will not include real customer, vendor, company, bank, invoice, or transaction data.

Planned files:

| File                  | Purpose                                                                   |
| --------------------- | ------------------------------------------------------------------------- |
| `vendors.csv`         | Synthetic vendor master data used for validation checks.                  |
| `purchase_orders.csv` | Synthetic purchase order data used for invoice matching.                  |
| `goods_receipts.csv`  | Synthetic goods receipt data used for three-way matching.                 |
| `invoices.csv`        | Synthetic invoice data used for process analysis.                         |
| `payments.csv`        | Synthetic payment execution data.                                         |
| `reconciliations.csv` | Synthetic reconciliation data used to compare invoice and payment status. |

## Purpose

This folder supports Phase 2 of the project by providing sample process data for SQL analysis, Python validation, KPI calculation, and visualization.

## Privacy Note

All data in this folder will be fully synthetic and created for portfolio use only.
